# JC System

Sistema de gestión empresarial para servicios técnicos y venta de equipos.

## Stack Tecnológico

- **Backend:** Node.js + Express + PostgreSQL + Prisma + JWT
- **Frontend:** React + Vite + Tailwind CSS
- **Notificaciones:** WhatsApp Web.js
- **Tareas programadas:** node-cron

## Requisitos previos

- Node.js v18 o superior
- PostgreSQL 17
- Git

## Instalación en nuevo equipo

### 1. Clonar el repositorio
```bash
git clone https://github.com/jcsystemcolombia-ux/jc-system.git
cd jc-system
```

### 2. Configurar el backend
```bash
cd backend
npm install
```
